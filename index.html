Elskom Official Website

To get the current runtime version visit: /runtime/version.txt

To download the current runtime and reference packages you must first take the latest version from that file, and use it to download the following gzip packages: - /runtime/packages/elskom-runtime-{version}-any.tar.gz - /runtime/packages/elskom-targeting-pack-{version}-any.tar.gz

Note: All of this should be done from the installer global tool that is published to nuget.org.

To properly install the runtime and reference packages one should use the global install tool published to nuget.org to install the entire workload. To be able to do that one should do dotnet tool install -g elskom.net.check. After that invoke elskom-check or elskom-check install to install, elskom-check update to update, or elskom-check uninstall to uninstall the workload.
